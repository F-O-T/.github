# F-O-T

Bibliotecas TypeScript prontas para produção, sistemas financeiros e ferramentas de criação de conteúdo com IA — construídos para o ecossistema brasileiro.

---

## Projetos

### [Montte](https://github.com/F-O-T/montte-nx)

Gestor financeiro moderno e de código aberto para controle de despesas pessoais e empresariais. Conta com gestão de transações, contas bancárias, boletos, estoque, orçamentos, metas financeiras e um motor de automação por regras visuais.

**Tecnologias:** React · ElysiaJS · Bun · tRPC · Drizzle ORM · PostgreSQL · BullMQ

### [Contentta](https://github.com/F-O-T/contentta-nx)

CMS de código aberto com edição potencializada por IA — como o Cursor, mas para criação de conteúdo. Editor baseado em Lexical com completações inline, edição agêntica, assistente de chat e planejamento de conteúdo com análise de SERP e descoberta de palavras-chave.

**Tecnologias:** React 19 · Lexical · Vercel AI SDK · Mastra · ElysiaJS · Bun · tRPC · Drizzle ORM · PostgreSQL

### [FOT Libraries](https://github.com/F-O-T/fot-libraries)

Coleção de bibliotecas TypeScript sem dependências externas cobrindo assinaturas digitais, documentos, cálculos financeiros e utilitários para o mercado brasileiro.

| Pacote | Descrição |
|--------|-----------|
| [`@f-o-t/e-signature`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/e-signature) | Assinatura PDF PAdES com conformidade ICP-Brasil e carimbos RFC 3161 |
| [`@f-o-t/digital-certificate`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/digital-certificate) | Certificados digitais A1 brasileiros (.pfx/.p12), XML-DSig e mTLS |
| [`@f-o-t/crypto`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/crypto) | PKCS#12, CMS/PKCS#7 SignedData, hashing e utilitários PEM |
| [`@f-o-t/asn1`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/asn1) | Codificação e decodificação ASN.1 DER |
| [`@f-o-t/money`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/money) | Operações monetárias com precisão BigInt e suporte ISO 4217 |
| [`@f-o-t/tax-calculator`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/tax-calculator) | Cálculo de impostos brasileiros |
| [`@f-o-t/rules-engine`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/rules-engine) | Motor de regras de negócio com avaliação de condições |
| [`@f-o-t/pdf`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/pdf) | Geração, análise e edição de PDF |
| [`@f-o-t/xml`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/xml) | Análise, manipulação, canonicalização XML e XPath |
| [`@f-o-t/brasil-api`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/brasil-api) | Integrações com APIs governamentais brasileiras |
| [`@f-o-t/ofx`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/ofx) | Parser e gerador OFX (Open Financial Exchange) |
| [`@f-o-t/qrcode`](https://github.com/F-O-T/fot-libraries/tree/master/libraries/qrcode) | Geração de QR codes em buffer PNG |
